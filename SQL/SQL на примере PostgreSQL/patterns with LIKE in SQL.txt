Конечно! Вот все основные паттерны, которые можно использовать с оператором `LIKE` в SQL:

1. **Поиск по точному совпадению**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE 'text';
   ```
   Этот паттерн ищет точное совпадение строки `text` в указанном столбце.

2. **Поиск по началу строки**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE 'text%';
   ```
   Этот паттерн ищет строки, которые начинаются с `text`.

3. **Поиск по окончанию строки**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE '%text';
   ```
   Этот паттерн ищет строки, которые заканчиваются на `text`.

4. **Поиск по подстроке**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE '%text%';
   ```
   Этот паттерн ищет строки, которые содержат `text` где угодно в строке.

5. **Поиск с одиночным подстановочным знаком**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE 't_xt';
   ```
   Символ `_` заменяет один любой символ. Этот паттерн ищет строки, где первая буква - `t`, третья - `x`, а четвертая - `t`.

6. **Поиск по множеству символов**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE '[abc]%';
   ```
   Этот паттерн ищет строки, которые начинаются с любого из символов `a`, `b`, или `c`.

7. **Поиск по диапазону символов**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE '[a-c]%';
   ```
   Этот паттерн ищет строки, которые начинаются с символа из диапазона `a` до `c`.

8. **Исключающий поиск**:
   ```sql
   SELECT * FROM table_name WHERE column_name LIKE '[^abc]%';
   ```
   Этот паттерн исключает строки, которые начинаются с символов `a`, `b`, или `c`.
