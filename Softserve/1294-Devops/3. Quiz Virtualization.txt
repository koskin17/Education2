What is the difference between full virtualization and para-virtualization? 
c. Full virtualization emulates hardware while para-virtualization uses modified guest operating systems.

В полном виртуализированном окружении гипервизор полностью эмулирует аппаратное обеспечение, позволяя запускать стандартные операционные системы без модификаций. What is the difference between a NAT network and a Bridged network in Oracle VirtualBox?
a. A Bridged network allows a virtual machine to access the host network while a NAT network does not.

В режиме Bridged сети виртуальная машина подключается к той же сети, что и хост, и может общаться с другими устройствами в этой сети, получая IP-адрес так же, как и физические устройства. В режиме NAT виртуальная машина получает доступ к интернету через хост, но не может напрямую общаться с другими устройствами в локальной сети.
