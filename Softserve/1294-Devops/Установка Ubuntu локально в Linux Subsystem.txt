Давай установим Ubuntu локально в WSL (Windows Subsystem for Linux) на твоём компьютере с Windows 10.

Шаги для установки Ubuntu в WSL на Windows 10:
Включение WSL:

Открой PowerShell или Командную строку от имени администратора. Для этого нажми Win + X, затем выбери Windows PowerShell (Admin) или Command Prompt (Admin).

Выполни команду для включения WSL:

powershell
wsl --install
Это установит WSL и загружает последнюю версию Ubuntu по умолчанию. Компьютер может перезагрузиться для завершения установки.

Проверка установки:

После перезагрузки открой PowerShell или Командную строку и введи:

powershell
wsl
Если Ubuntu уже установлен, ты увидишь приглашение терминала Ubuntu. Если нет, следуй следующим шагам.

Установка конкретного дистрибутива (опционально):

Если ты хочешь установить конкретную версию Ubuntu, ты можешь сделать это через Microsoft Store.

Открой Microsoft Store, введи в поиске Ubuntu и выбери нужную версию (например, Ubuntu 20.04 LTS).

Нажми Get (Получить), затем Install (Установить).

Запуск Ubuntu:

После установки открой Ubuntu из меню "Пуск" или через PowerShell командой:

powershell
wsl
Следуй инструкциям по настройке, включая создание нового пользователя и пароля.

Проверка работы Ubuntu в WSL
Запуск терминала Ubuntu:

Открой PowerShell или Командную строку и введи:

powershell
wsl
Ты должен увидеть приглашение терминала Ubuntu.

Проверка версии Ubuntu:

Введи команду:

bash
lsb_release -a
Это покажет информацию о версии установленного дистрибутива Ubuntu.
