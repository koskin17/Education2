
Вопрос 1: Какой тип подзапроса выполняется один раз для всего внешнего запроса?
Ответ: d. Некоррелированный подзапрос
Объяснение: Некоррелированный подзапрос выполняется один раз для всего внешнего запроса и не зависит от данных внешнего запроса.

Вопрос 2: Какой оператор используется для сравнения одного значения с результатом подзапроса?
Ответ: a. IN
Объяснение: Оператор IN используется для проверки, принадлежит ли значение множеству значений, возвращаемых подзапросом.

Вопрос 3: Что возвращает оператор EXISTS, если подзапрос возвращает хотя бы одну строку?
Ответ: d. True
Объяснение: Оператор EXISTS возвращает значение TRUE, если подзапрос возвращает хотя бы одну строку.

Вопрос 4: Сопоставьте оператор подзапроса с его функцией:
Ответ 1: EXISTS — проверяет, возвращает ли подзапрос хотя бы одну строку.
Ответ 2: ANY — сравнивает значение с результатом подзапроса, который возвращает несколько строк.
Ответ 3: SELECT — ключевое слово, которое обычно используется для введения подзапроса.
Ответ 4: NOT EXISTS — проверяет, возвращает ли подзапрос нулевые строки.

Вопрос 5: Какой тип подзапроса оценивается для каждой строки, обрабатываемой внешним запросом?
Ответ: b. Коррелированный подзапрос
Объяснение: Коррелированный подзапрос оценивается для каждой строки внешнего запроса и зависит от данных внешнего запроса.

Вопрос 6: Какое ключевое слово используется для исключения дубликатов строк в результате подзапроса?
Ответ: a. DISTINCT
Объяснение: Ключевое слово DISTINCT используется для исключения дублирующихся строк в результате подзапроса.

Вопрос 7: Сопоставьте тип подзапроса с его описанием:
Ответ 1: Скалярный подзапрос — подзапрос, который возвращает один столбец и одну строку.
Ответ 2: Некоррелированный подзапрос — подзапрос, который выполняется один раз для всего внешнего запроса.
Ответ 3: Вложенный подзапрос — подзапрос, который определяется в WHERE-условии внешнего запроса.
Ответ 4: Коррелированный подзапрос — подзапрос, который выполняется для каждой строки внешнего запроса.

Вопрос 8: Какое из следующих утверждений истинно относительно подзапросов?
Ответ: c. Включает использование внутреннего и внешнего запроса
Объяснение: Подзапросы всегда включают два запроса — внешний и внутренний.

Вопрос 9: Какой из следующих подзапросов является коррелированным?
Ответ: a. Это подзапрос, который содержит ссылку на таблицу, которая также встречается во внешнем запросе.
Объяснение: Коррелированный подзапрос зависит от внешнего запроса и содержит ссылки на его таблицы.

Вопрос 10: Подзапрос в SQL SELECT-запросе заключается в:
Ответ: a. Круглые скобки -- (...)
Объяснение: Подзапросы всегда заключаются в круглые скобки.
