/*
UNION позволяет добавить одну таблицу вниз другой, т.е. добавление таблиц друг под друга, вниз
Для этого таблицы должны иметь одинаковые колонки, чтобы данные подставились снизу.
При этом для названий колонок можно использовать алиасы.
Если же в колонке встретятся одинаковые значения, то UNION удалит дубликаты.
Если же дубликаты строк нужны, то к UNION добавляется ALL - UNION ALL

Если нужно объединить одинаковый столбец двух таблиц, но добавить тип или признак того, что это такое, о чём данные,
то используется тип:
SELECT city, 'customer city' AS type FROM customers
UNION
SELECT city, 'supplier city' FROM suppliers
ORDER BY city DESC

В результате будет добавлен столбец type с указанием типов, которые прописаны
*/
