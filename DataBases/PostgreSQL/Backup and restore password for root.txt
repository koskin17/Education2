Да, вы можете узнать и изменить пароль root пользователя после установки PostgreSQL. Вот шаги, которые помогут вам сделать это:

1. **Создайте резервную копию файла `pg_hba.conf`**: Найдите этот файл в директории данных PostgreSQL (например, `C:\Program Files\PostgreSQL\16\data` на Windows) и скопируйте его в другое место или переименуйте его (например, `pg_hba.conf.bk`).

2. **Измените параметры в файле `pg_hba.conf`**: Откройте файл `pg_hba.conf` и измените все локальные соединения с `scram-sha-256` на `trust`. Это позволит вам войти в сервер PostgreSQL без пароля.

3. **Перезапустите сервер PostgreSQL**: Если вы используете Windows, вы можете перезапустить PostgreSQL через службы. В противном случае, выполните команду `pg_ctl restart -D "C:\Program Files\PostgreSQL\16\data"` из командной строки с правами администратора.

4. **Подключитесь к серверу PostgreSQL**: Используйте любой инструмент, такой как `psql` или `pgAdmin`, чтобы подключиться к серверу PostgreSQL. Теперь вам не потребуется вводить пароль.

5. **Установите новый пароль**: Выполните команду `ALTER USER postgres WITH PASSWORD 'new_password';` для установки нового пароля.

6. **Восстановите файл `pg_hba.conf`**: Верните файл `pg_hba.conf` в исходное состояние, перезапустите сервер PostgreSQL и подключитесь к нему с новым паролем.

Если у вас возникнут какие-либо трудности или вопросы, не стесняйтесь обращаться за помощью!
